# ELEX DAVV Portal

A comprehensive Previous Year Question Papers portal for ELEX DAVV (School of Electronics, Devi Ahilya Vishwavidyalaya) students.

## Features

- **Subject-wise Organization**: Browse question papers by semester and subject
- **Course Support**: BSc Electronics and Integrated MTech Electronics
- **Smart Search**: Search by subject name or code
- **Responsive Design**: Works on all devices
- **Dark Theme**: Modern dark interface for better readability

## Technology Stack

- **Frontend**: React + TypeScript + Vite
- **Styling**: Tailwind CSS + Shadcn/ui
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Getting Started

### Prerequisites

- Node.js (v18 or higher)
- npm or yarn

### Installation

1. Clone the repository:
```bash
git clone <repository-url>
cd elexvault-main
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

4. Open [http://localhost:8080](http://localhost:8080) in your browser.
